Getting Started with the MakerNest Platform

Software2026-04-09

You will learn

After reading, you can complete the account settings independently, understand the structure of the home page, and download the first model file smoothly.

Registration and information

After logging in with an email or a third-party account, it is recommended to complete the avatar and profile in the "Personal Data" to facilitate community interaction and verification of subsequent points tasks.

!
Tips

Nicknames and briefings are displayed next to reviews and works, avoiding the use of illegal or infringing descriptions.

Home and Search

The home page aggregates recommended streams and thematic entries, and the top search supports titles and tags of works; the results page can be further filtered by material and processing time.

Downloads and File Type

The work details page provides the design file and the processing file download entrance, if the author sets the permission condition, please confirm the authorization scope in the page first and then download.
